Présentation Javascript

Le javascript est un langage de programmation qui peut être exécuté dans un navigateur : il va permettre de dynamiser la page.

Il est exécuté côté client : on exécute du code sans passer par un serveur.

Pour écrire du JS dans une page web, il faut une balise script. On peut écrire le code directement dedans, ou alors cette balise peut inclure un fichier JS externe.
On place en général le code javascript en bas de la page, car il est bloquant. Et donc on affiche le HTML/CSS avant de commencer à exécuter du JS.
