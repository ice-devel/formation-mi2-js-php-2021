Variables Javascript

Comme pour tout langage de programmation, le JS peut utiliser des variables : Une variable, c'est une emplacement en mémoire où l'on peut stocker une (ou plusieurs) valeurs, représentée dans le programme par un nom.

Le langage JS est un langage faiblement typé : il n'y a pas obligation à type ses variables. JS le fait pour nous à l'exécution. Les types primitifs en javascript :

  • Chaines de caractères (string)
  • Nombres (number) : entier + nombre à virgule
  • Boolean (boolean) : vrai ou faux
